# Alena's 12th Birthday Profile 🎀💜

A fun, interactive birthday celebration website for Alena's 12th birthday!

## Features

- ✨ Animated hero section with gradient text
- 🎂 Interactive birthday cake with candle lighting animation
- ⏳ Live countdown timer to birthday
- 🎀 Responsive design optimized for all devices
- 📸 Photo gallery with lightbox viewer
- 💌 Birthday wishes wall
- 🎉 Confetti animations
- 🎨 Purple bow-core themed design

## Pages

- **Home** - Main hero section with countdown
- **Countdown** - Live timer showing days, hours, minutes, seconds
- **Cake** - Interactive birthday cake with animation
- **Fast Facts** - Fun facts about the birthday girl
- **Gallery** - Photo gallery of memories
- **Wishes** - Birthday wishes wall where friends can leave messages

## Technology

- Pure HTML5
- CSS3 (animations, gradients, flexbox, grid)
- Vanilla JavaScript (no frameworks)
- LocalStorage for persistent wishes

## Deployment Options

### GitHub Pages

1. Create a new repository on [GitHub](https://github.com/new)
2. Name it: `alena-birthday-profile`
3. Run these commands:

```bash
git remote add origin https://github.com/YOUR_USERNAME/alena-birthday-profile.git
git branch -M main
git push -u origin main
```

4. Go to repository Settings → Pages
5. Set source to `main` branch
6. Your site will be live at: `https://YOUR_USERNAME.github.io/alena-birthday-profile/`

## Local Development

Simply open `index.html` in your browser. No build process needed!
